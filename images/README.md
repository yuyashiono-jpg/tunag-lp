# images/ — 素材リスト

このフォルダに含まれる画像素材の一覧です。サンプルLP用のプレースホルダーとして同梱していますが、購入者が自社のサービスに合わせて差し替えることを想定しています。

---

## 画像一覧

### hero-visual.png
- **用途**: ヒーローセクションのメインビジュアル（右側）
- **推奨サイズ**: 幅 800〜1200px / 高さ 自動（縦横比を保持）
- **トーン**: 白背景ベース。プロダクトUIモック or ビジネスシーンのイラスト
- **注意**: ブランドグラデーション(青→アクア)の上に載るので、白背景の素材が映える

### solution_01.png / solution_02.png / solution_03.png
- **用途**: Solutionセクションの3つのポイント画像（左側・大きめ）
- **推奨サイズ**: 幅 1200px 以上 / デスクトップで height: 520px + object-fit: cover
- **トーン**: サービスの具体的な画面キャプチャ、または機能を説明するイラスト
- **注意**: 角丸(16px)でクリップされるため、四隅に重要な要素を置かない

### issues_icon_1.png / issues_icon_2.png / issues_icon_3.png
- **用途**: Issuesセクションのカード上部画像
- **推奨サイズ**: 幅 600〜800px / 高さは横幅の50〜70%程度
- **トーン**: ブランドカラー(青系)のアイソメトリック3Dイラスト。課題を視覚的に表現
- **注意**: カードの上端・左右端にブリードする配置。余白は少なめに

### users_1.png / users_2.png / users_3.png
- **用途**: お客様の声(Voice)セクションのアバター
- **推奨サイズ**: 200x200px 以上の正方形（CSSで86pxの円形にクリップ）
- **トーン**: 人物の顔写真 or ビジネスパーソンのイラスト
- **注意**: border-radius: 50% で円形表示。顔が中央に来る構図で

### logo.png
- **用途**: 導入企業ロゴカルーセル
- **推奨サイズ**: 高さ 80px 程度 / 幅は自然なアスペクト比で
- **トーン**: 企業ロゴの原色そのまま（加工禁止・白背景固定）
- **注意**: 5社分以上を用意し、ファイル名を logo_1.png 〜 logo_5.png 等で分ける

### pattern_grid.png
- **用途**: セクション背景の薄いグリッドパターン（::before擬似要素で opacity: 0.05）
- **推奨サイズ**: 300x300px のタイル用パターン
- **トーン**: グレーの細い線。目立たない程度
- **注意**: テンプレートに同梱済み。差し替え不要
